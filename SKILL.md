- 去重

- 交叉验证（同一数据点从 2-3 个来源确认）
- **精度要求**：保留小数点后两位（如 65.76亿元，不是 65亿）

### Step 4: 填充报告模板

按十大模块逐一生成，遵循飞书卡片格式规范。

### Step 5: 质量检查

- 数据一致性
- 格式合规性（飞书卡片格式）
- 风格符合度（基金从业人员风格）
- 精度检查（所有金额保留两位小数）

### Step 6: 推送输出

发送报告，以专业机构研报格式呈现。

## 基金从业人员风格指南

### 语调特征

- **主动出击**：不用"建议关注"，用"超配！"、"加仓！"
- **带点叛逆**：敢于说"别碰"、"快跑"、"纯炒作"
- **数据驱动**：每句话都有数据支撑
- **口语化**：像和朋友聊天，不是写论文
- **热辣点评**：每个热点都要有一句犀利的机构点评

### 示例对比

❌ 官方腔："建议投资者谨慎对待高估值板块"
✅ 机构风："高位股别碰！富国银行都喊话获利了结了，你还恋战什么？"

❌ 模糊表述："可能出现回调风险"
✅ 机构风："SOXL遭最大规模抛售，恐高情绪明显，随时可能回调10%！"

❌ 没有点评："算力租赁概念涨幅居前"
✅ 机构风："算力租赁这波太猛！AI需求井喷，数据中心供不应求，继续超配！"

## 数据优先级

### P0（必须获取）

- 三大/五大指数收盘数据（点位、涨跌幅、成交额）
- 板块资金流向（TOP5 流入/流出）- **精确到小数点后两位**
- 北向/南向/外资净流入数据 - **精确金额**
- **地缘核心数据**：美伊谈判进展、OPEC/油价动态
- **宏观核心数据**：WTI原油价格、美元/离岸人民币汇率
- **大宗商品**：黄金、铜、钨等与A股板块联动品种

### P1（重要补充）

- 热点事件 TOP5（催化剂 + 个股表现）
- 散户情绪指标（涨停家数、炸板率、讨论热度）
- **多平台讨论热度数据**：
  - 东财论坛（帖子数、情感倾向）
  - 雪球（讨论量、热门话题）
  - 淘股吧（连板股讨论、追涨情绪）
  - 同花顺（评论区情绪）
  - 富途牛牛（港股讨论热度）
  - 老虎社区（中概股讨论）
- 机构持仓变化（对冲基金、共同基金）
- **亚洲三市外资数据**（日本/韩国精确到亿美元）
- **美联储降息预期**（CME FedWatch Tool 数据）
- **中国宏观数据**（Q1 GDP、CPI、PPI）
- **全球宏观周期定位**（美国/中国/日本/韩国对比表）
- **大宗商品与汇率联动数据**（铜/黄金/原油/人民币）

### 美股专项数据（P0）

- **VIX 恐慌指数**（当前值、涨跌幅、历史分位）→ **优先调用 vix-index skill**
- VIX9D（9日波动率）
- VVIX（VIX的波动率，>80 预警）
- VIX 与美股走势背离分析
- Put/Call Ratio（期权成交量比）
- Skew 指标（尾部风险）

> **注意：** VIX 数据**必须优先调用 vix-index skill**，只有当 skill 失败时才使用 online-search 作为降级方案。

### P2（增值内容）

- 跨市场对比（亚洲三市资金迁移）
- 分析师评级调整
- 期权/衍生品信号
- **资金迁移可视化路径图**
- **地缘传导路径图**（美伊谈判 → 油价 → 纳指 → A股开盘）
- **全球资金流向与资产配置热力图**

### A股港股报告数据获取（6组并行搜索）

```
搜索 1: 2026年4月16日 A股收盘 上证指数 深证成指 创业板指 成交额
搜索 2: 2026年4月16日 A股板块 资金流向 净流入 净流出 TOP5
搜索 3: 2026年4月16日 A股 涨停 连板 封板率 晋级率
搜索 4: 2026年4月16日 港股收盘 恒生指数 恒生科技 南向资金
搜索 5: 2026年4月16日 北向资金 净流入 沪股通 深股通
搜索 6: 2026年4月16日 日韩股市 日经225 韩国KOSPI 收盘
```

### 数据获取最佳实践

1. **并行搜索**：同时发起多个 ProSearch 查询，减少等待时间
2. **日期覆盖**：搜索 T 日和 T-1 日数据，确保完整性
3. **关键词组合**：
   - 指数：`收盘 + 指数名称 + 涨跌幅`
   - 资金：`板块 + 资金流向 + 净流入/流出`
   - 情绪：`涨停 + 连板 + 封板率`
   - 外资：`北向资金/南向资金 + 净流入`
4. **数据验证**：交叉比对多个搜索结果，取最精确数值
5. **降级方案**：若 ProSearch 失败，使用 online-search 作为备选

## 五维分析模型（升级为六维）

1. **指数维度** → 市场整体强弱
2. **资金维度** → 主力/外资/散户三方博弈
3. **板块维度** → 结构性机会识别
4. **情绪维度** → 过热/过冷逆向信号（**含VIX恐慌指数**）
5. **宏观维度** → 政策/地缘/数据催化
6. **地缘与宏观维度** → 美伊谈判/美联储/大宗商品/汇率/全球资金流向

> ⚠️ **宏观地缘是必评维度**，每份报告第9.5节必须填写，不可省略。
